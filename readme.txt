== Description ==
Plugin to make a semantics cocoon on wordpress by simply loading a mindmap file.

== Installation ==

1. Download mmtowp.zip from the "download" link on the web page where you're viewing this.
2. Decompress the file contents.
3. Upload the mmtowp folder to your WordPress plugins directory (/wp-content/plugins/).
4. Activate the Mindmap to WordPress plugin from the WordPress back-office.

== Usage ==

- Import your mindmap file
- Check all pending pages that have been created
